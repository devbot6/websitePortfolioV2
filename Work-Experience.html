402nd Software Engineering Group Summer Internship (1st Year) 2023 Summer

I completed my 2023 summer internship at Robbins Air Force Base 402nd Software Engineering Group where I studied a broad variety of skills such as version control managment using github, python fundamentals, navigating the terminal, and getting comfortable with raspberry Pis.

 Bandit OTW

We practiced navigating the terminal using a website game called Bandit where we navigated the terminal to find passwords to pass a level.

Github

We learned how to use git and github to manage our code across multiple people. By using branches, master branch and pull requests.

Python

We learned the fundamentals of python and created projects such as a BMI calculator, calculator, grocery list, and battleship.

Raspberry PI

We also learned how to set up a PI and SSH into the PI to put software on the physical PI.

Soft Skills

We also learned a variety of soft skills such as communication, public speaking, teamwork, written communication, punctuality, leadershipm and problem solving.
