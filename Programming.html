Titanic Dead or Not AI Program

This is a program that uses Machine Learnining to determine if a passenger of the Titanic survied or died based off of attributes such as fare price, sex, class, and age. This program uses a model called RandomForestClassifier that essentially uses a tree diagram to make decisions, and based on those decisions it will put it in the class survived or not survived. So the model reads data through a csv file using pandas, that data is then modiefied based on the attributes i want and don't want through the drop function whihc essentially drops any attributes and doesn't consider them. The model requires everything to be numerical so the sex must be modied to 1's and 0's in order for the model to read it so that is done as well. The data is then split into 80% testing data and 20% evaluating data. This will allow the model to see patterns in the data and make prediction on whether people lived or died based on that.

Titanic Dead or Not AI Repo: https://github.com/devbot6/TitanicDeadOrNotAi

Python Battleship

This is a project I did for my first year summer internship for the Robins Air Force Base 402nd Software Engineering Group. It is essentially a terminal based game that is resembled to be battlehsip it was created by me Devon Hulse as well as my partner Jayden Wright. The way it works is it uses 2 lists, one for the user's ships and one for the computer's ships. The computer's ships are randomly stored in the 2d array and the user is prompted to guess, they have 5 chances to get it and if they don;t the computer gets a chance to shoot their ships that are hand placed by the user if the computer happens to land on one they win.
